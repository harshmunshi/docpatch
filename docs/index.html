02 / Integration

One import, five stages.

Wire DocPatch into your agent loop in minutes. Use the convenience API or assemble the pipeline yourself.

A

Convenience API

Two functions cover the full pipeline. open_doc parses a file; edit_doc locates, patches, and splices.

python 
from docpatch import open_doc, edit_doc
from docpatch.models.anthropic import AnthropicClient

# parse a file into an addressable DocTree
tree = open_doc("spec.md")

# wire up any ModelClient-compatible client
model = AnthropicClient(model="claude-haiku-4-5-20251001")

# locate the right node, send only that node to the model,
# validate the result, and splice it back — losslessly
patched_tree = edit_doc(
    tree,
    "Tighten the introduction section",
    model=model,
)

# serialize back to bytes — unchanged regions are untouched
from docpatch.parsers import get_parser, detect_format
parser = get_parser(detect_format("spec.md", None))
output_bytes = parser.serialize(patched_tree)
open("spec.md", "wb").write(output_bytes)
B

Manual pipeline

For fine-grained control — custom locator thresholds, multiple patches, dry-run inspection, or swap in your own model client.

python 
from docpatch.parsers.markdown import parse, serialize
from docpatch.locator import CompositeLocator
from docpatch.patcher.replace import ReplaceOperation
from docpatch.splicer import splice, validate_splice
from docpatch.models.mock import MockModelClient

# 1. Parse
data = open("report.md", "rb").read()
tree = parse(data)

# 2. Locate — symbolic first, semantic fallback
model = MockModelClient(response="Rewritten content goes here.")
locator = CompositeLocator(model=model, threshold=0.6)
result = locator.locate(tree, "Replace the executive summary")

if not result.node_ids:
    raise ValueError(f"Not found: {result.candidates}")

# 3. Patch — only the target subtree reaches the model
op = ReplaceOperation(max_retry=3)
patch = op.apply(tree, result.node_ids[0], "Tighten and clarify", model)

# 4. Splice — deterministic, no model call
new_tree = splice(tree, [patch])
validation = validate_splice(tree, new_tree, [patch])
assert validation.ok, validation.error

# 5. Serialize — unchanged regions return their exact original bytes
open("report.md", "wb").write(serialize(new_tree))
C

Command-line interface

The docpatch edit command runs the full pipeline in one invocation. Use --dry-run to preview before writing.

shell 
# uses MockModelClient by default — safe for testing
docpatch edit README.md "Tighten the introduction paragraph"

# preview without writing
docpatch edit spec.md "Add a note about rate limits" --dry-run
shell 
# set key in env; --model selects provider
export DOCPATCH_API_KEY="sk-ant-..."
docpatch edit spec.md "Rewrite the executive summary" \
  --model anthropic

# OpenAI
export DOCPATCH_API_KEY="sk-..."
docpatch edit data.json "Update the version field to 2.0" \
  --model openai
shell 
# write to a different path (original untouched)
docpatch edit spec.md "Tighten section 2" \
  --model anthropic \
  --out spec-edited.md

# chain: edit, then edit the result
docpatch edit spec.md "Tighten intro" --out /tmp/pass1.md
docpatch edit /tmp/pass1.md "Add TL;DR" --out spec-final.md
D

Bring your own model

DocPatch is model-agnostic. Any object that implements the ModelClient protocol works — including local models, proxies, and test doubles.

python 
from docpatch.models.base import ModelClient, ModelResponse

# Satisfy the Protocol — no inheritance required
class MyLLM:
    def complete(self, prompt: str, *, max_tokens: int) -> ModelResponse:
        raw = my_local_llm.generate(prompt, max_tokens=max_tokens)
        return ModelResponse(
            text=raw.text,
            tokens_in=raw.usage.prompt,
            tokens_out=raw.usage.completion,
            model_id="my-local-model",
        )

# Drop it in anywhere a ModelClient is expected
patched = edit_doc(tree, "Rewrite intro", model=MyLLM())
anthropic openai litellm any Protocol-compatible object
E

Inspect the DocTree

Every node has a deterministic ID, a BLAKE3 fingerprint, and a raw_span back to the original bytes. Traverse, diff, or serialize individual subtrees.

python 
from docpatch.parsers.markdown import parse

tree = parse(open("README.md", "rb").read())

# walk every node depth-first
for node in tree.walk():
    print(node.id, node.type, node.fingerprint[:8])

# look up a node by its deterministic ID
node = tree.get_node("heading:document:root/introduction#0")

# list the ancestor path (breadcrumb for prompt context)
ancestors = tree.ancestors(node.id)

# get the heading skeleton (used by SemanticLocator)
# — only headings, never paragraph content
skeleton = tree.heading_skeleton()
# → "# Introduction\n## Background\n## Results\n"

05 / Speed

Edit time is O(1) in document size.

LLM latency is driven by output tokens. A naive edit re-emits the whole document. DocPatch emits only the patched node — the same ~150 tokens whether the document is 2 pages or 200 pages.
